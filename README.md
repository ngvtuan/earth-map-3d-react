A Component for Using Lumera SDK-JS in Next.js (Lumera Hub)

## Installation

You can install this library via NPM or YARN.

### NPM

```bash
npm i earth-map-3d-react

```

### YARN

```bash
yarn add earth-map-3d-react
```
